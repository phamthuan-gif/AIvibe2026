Danh sách việc cần làm

  1. Tạo trang HTML cơ bản (<html>, <head>, <body>)
  2. Vẽ mèo bằng các khối <div>: đầu, thân, 4 chân, đuôi
  3. Dùng CSS để tạo hình tròn, hình oval, màu lông
  4. Thêm cảm xúc: cười / tức giận (đổi mắt + miệng)
  5. Thêm tư thế: đứng / nằm / đi (animation)
  6. Thêm nút bấm bằng JavaScript để điều khiển
  7. Tự thử: đổi màu mèo trong file style.css (tìm --fur)
  8. Tự thử: thêm nút “Ngủ” (mắt nhắm) — hỏi mình nếu cần giúp!

Bạn đang học gì?

  • HTML: các thẻ tạo cấu trúc (mèo gồm nhiều div)
  • CSS: màu, kích thước, vị trí, hiệu ứng đi
  • JavaScript: khi bấm nút → đổi class trên mèo
